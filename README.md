# vue-mobil-ipad

## 说明
```
(搭本框架是为了适配iPad横屏模式和兼容大部分手机的横屏模式)
UI库配置了京东的开源库
px自动转换为vw
UI设计图推荐iPad的尺寸(1024*768)的2倍图
```
